# star-ams
Repo for the athlete management information system being built

Project scope

	- Objective: successfully launch an athlete management mobile and web application.
	- Deliverables: Mobile app with acceptable user interface and a web app that provides more tools and features to manage athletes.
	- Exclusions: Technology upgrades and intense maintenance post-launch
	- Constraint: Timeline of two years max. Stretch goal of 1.5 years.


Overview of technology
 - React native Frontend
 - Java Backend
 - MSSQL DB
